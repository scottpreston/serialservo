API

GET - /move/{pin}/{pos}

  • pin - 0 to 7, the pin it's plugged in on the SSC.
  • pos - 0 to 255. 0 Full CCW, 127 - stopped, 255 Full CW.

GET - /nav/{left_motor}/{right_motor}

  • left_motor - 0 to 255. 0 Full CCW, 127 - stopped, 255 Full CW.
  • right_motor - 0 to 255. 0 Full CCW, 127 - stopped, 255 Full CW.
